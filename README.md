# Gestor de Tareas de Equipo

Aplicación de consola en Java para gestionar una lista de tareas de equipo.

## Funcionalidades

- Añadir tareas con una descripción.
- Listar todas las tareas registradas.
- Marcar tareas como completadas.

## Estructura del proyecto

- `Tarea.java`: representa una tarea individual.
- `GestorTareas.java`: contiene la lógica de gestión de la lista de tareas.
- `Main.java`: muestra el menú y gestiona la interacción con el usuario.

## Uso

Compilar y ejecutar `Main`.  
Desde el menú se pueden añadir tareas, listarlas y marcarlas como completadas.
